## Background

Fashion-MNIST is a public dataset of Zalando Researchâ€”consisting of a training set of 60,000 example photos of articles of clothing and a test set of 10,000 examples. 

Each example is a 28x28 grayscale image, associated with a clothing category (labels) from 10 types of clothing. 

The original dataset can be found here:
https://www.kaggle.com/sonalidasgupta95/fashion-mnist-dataset-exploration/data


## Labels (Clothing Types)

Each training and test example is assigned to one of the following labels:

- T-shirt/top
- 1 Trouser
- 2 Pullover
- 3 Dress
- 4 Coat
- 5 Sandal
- 6 Shirt
- 7 Sneaker
- 8 Bag
- 9 Ankle boot
